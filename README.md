In order to get this function youll have to jump through a few hoops.

1. Youll have to download Node.js: 
https://nodejs.org/en/download 

2. Go to the Visual Studio Terminal and type in: 
npm install

3. Type in the terminal: 
npm init -y

4. Type in the terminal: 
npm install puppeteer

5. Run the Server.Js

6. Type into google: 
http://localhost:3000/


---------------------------------------------------------------------------------------


In the case it dosent work that way for you on step 2 which is common

1.1 Go to powershell and type in: 
Set-ExecutionPolicy RemoteSigned

1.2 Type in the capital letter: 
Y

Continue from step 2

---------------------------------------------------------------------------------------

1.3 If it still doesnt work:
Download it as a zip

1.4 Open a new empty folder

1.5 Do step 1.1; 1.2; 2; 3; 4

1.6 Then go to the Zip folder and get the public folder and server.js file and move it from the zip to your new empty file

Continue from step 5
